# LINK ON FIT — 관리자 웹 페이지 기획

## 개요
Flutter 앱(linkfit)과 연동되는 헬스장 운영용 웹 관리자 페이지.  
대상: 헬스장 원장·관리자 / 스택: 미정(React 또는 Next.js 권장)

---

## 기능 목록

### 1. 헬스장 이용정보 관리
- [ ] 운영시간 설정 (요일별 오픈·마감 시간)
- [ ] 휴일 설정 (공휴일, 임시 휴관일 캘린더)
- [ ] PT권 잔여 횟수 현황 (회원별 조회)
- [ ] 공지사항 배너 등록·수정·삭제

---

### 2. 직원(트레이너) 관리
- [ ] 트레이너 목록 조회 (사진, 이름, 자격증, 경력)
- [ ] 트레이너 계정 생성·수정·비활성화
- [ ] 앱 트레이너 계정 지정 (회원 ↔ 트레이너 역할 전환)
- [ ] 출근 여부 현황판 (오늘 출근/미출근 트레이너 목록)
- [ ] 출근 기록 조회 (날짜별 출퇴근 이력)

---

### 3. 회원 관리
- [ ] 회원 목록 조회·검색·필터 (이름, 등급, 이용권 상태)
- [ ] 회원 상세 정보 조회·수정 (연락처, 가입일, 담당 트레이너)
- [ ] 구독 등급 관리 (Basic / Light Fit / Regular Fit / Intensive Fit)
- [ ] 헬스장 이용권 부여·회수·만료일 설정
- [ ] 상담 기록 조회·추가 (상담 날짜, 내용, 담당 트레이너)
- [ ] 회원 탈퇴 처리

---

### 4. 이용권(티켓) 관리
- [ ] 원포인트 티켓 목록 조회 (보유자, 만료일, 사용 여부)
- [ ] 피드백 티켓 목록 조회
- [ ] 티켓 구매 처리 (관리자가 직접 결제 후 발급)
- [ ] 티켓 회원 또는 트레이너에게 지급·회수
- [ ] 지급 이력 조회 (발급일, 지급 대상, 처리자)

---

### 5. 수업 관리
- [ ] 수업 일정 캘린더 (전체 PT·OT·원포인트 일정 한눈에 보기)
- [ ] 수업별 상세 조회 (회원, 트레이너, 날짜·시간, 수업 유형)
- [ ] 수업 상태 관리 (예정 / 진행중 / 완료 / 취소)
- [ ] 원포인트 레슨 신청서 조회 및 승인·거절
- [ ] OT 배정 (신규 회원 → 트레이너 매칭)

---

### 6. 출석 관리
- [ ] 회원별 출석 현황 조회 (이번 달 출석률)
- [ ] 루틴 실행 이력 조회 (날짜별 완료·미완료)
- [ ] 장기 미출석 회원 알림 목록 (ex. 7일 이상 미출석)
- [ ] 출석 통계 (주간·월간 출석 차트)

---

### 7. 매출 및 상품 관리
- [ ] 매출 대시보드 (오늘 / 이번 달 / 누적 매출)
- [ ] 구독권 판매 현황 (등급별 가입자 수, 신규·해지 추이)
- [ ] 티켓 판매 현황 (원포인트·피드백 판매량)
- [ ] 상품 목록 관리 (구독 플랜 가격·혜택 수정)
- [ ] 결제 내역 조회·환불 처리
- [ ] 매출 리포트 엑셀 내보내기

---

### 8. 메시지(공지·이벤트)
- [ ] 전체 공지 발송 (앱 내 쪽지 → 전체 회원)
- [ ] 트레이너 전용 공지 발송
- [ ] 특정 회원 그룹 발송 (등급별, 담당 트레이너별)
- [ ] 이벤트 공지 등록 (제목, 내용, 이미지, 기간)
- [ ] 발송 이력 조회 (발송일, 수신 대상, 내용)

---

## 공통 요구사항

- 관리자 로그인 (이메일+비밀번호, JWT 인증)
- 역할 구분: 슈퍼관리자(원장) / 일반관리자(데스크 직원)
- 반응형 레이아웃 (PC 우선, 태블릿 지원)
- 기존 백엔드 API(`localhost:17577`) 연동
- 다크/라이트 테마 (앱 디자인 시스템 참고)

---

## 백엔드 추가 필요 API (예상)

| 기능 | 메서드 | 엔드포인트 |
|------|--------|------------|
| 출근 현황 조회 | GET | `/admin/trainers/attendance` |
| 트레이너 계정 역할 변경 | PATCH | `/admin/users/{id}/role` |
| 수업 일정 전체 조회 | GET | `/admin/schedules` |
| 원포인트 신청 목록 조회 | GET | `/admin/onepoint/requests` |
| 원포인트 신청 승인·거절 | PATCH | `/admin/onepoint/requests/{id}/status` |
| 티켓 지급 | POST | `/admin/tickets/grant` |
| 매출 통계 | GET | `/admin/sales/stats` |
| 전체 공지 발송 | POST | `/admin/messages/broadcast` |
| 회원 상담 기록 추가 | POST | `/admin/members/{id}/consultations` |
| 장기 미출석 회원 조회 | GET | `/admin/members/inactive` |
